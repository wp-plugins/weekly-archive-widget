== Description ==

Create a widget that displays the archives by week

== Installation ==

1. Copy the plugin files to <code>wp-content/plugins/</code>

2. Activate the plugin from Plugins page

3. Go to Appearance -> Widgets

4. Look for Weekly Archive and drag to appropriate sidebar

== Frequently Asked Questions ==

= How do I change the day of the week the archive starts on? =

Go to Settings -> General and change the value of <b>Week Starts On</b>.
